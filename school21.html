21 School projects

В январе 2025 года я проходил отборочный интенсив, который длился в месяц. В феврале поступил на основное обучение, которое стартовало 23 апреля. На интенсиве мы активно работали с bash, linux, git и языком C. На основном обучении на данный момент я выполнил первый проект по ветке(направлению) DevOps. В данном проекте я работал с ОС Ubuntu 20.04 Server LTS на виртуальном машине в Virtual Box.

Приобретённые навыки:

1. Администрирование Linux-серверов:

  • - Установка и базовая настройка ОС.
  • - Управление пользователями и группами.
  • - Настройка сети (статический/DHCP, шлюз, DNS).

2. Работа с командной строкой:

  • - Использование консольных утилит для мониторинга и диагностики.
  • - Навыки работы с текстовыми редакторами.

3. Безопасность:

  • - Настройка SSH (смена порта, проверка логов авторизации).
  • - Разграничение прав через sudo.

4. Оптимизация и обслуживание:

  • - Обновление системы.
  • - Синхронизация времени.
  • - Управление задачами через Cron.

5. Анализ и решение проблем:

  • - Чтение системных журналов.
  • - Диагностика сетевых проблем.
